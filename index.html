Zero-Trust Architecture

Your privacy is not a feature, it's the foundation. We use a layered approach to security.

  • **Bcrypt Hashing**: Passwords stored using the most secure hashing standard.
  • **Data Isolation**: Each user's data is strictly linked to their session ID and MongoDB user_id.
  • **Session Management**: Protected by Flask server-side session management.
  • **Secure APIs**: All data retrieval requires active, validated login session.
